異核二原子分子の分子軌道の作り方を説明します。

## 基本的な手順

1. **原子軌道のエネルギー準位を確認**
   - 各原子の価電子軌道のエネルギーを比較
   - 電気陰性度が高い原子ほど軌道エネルギーが低い

2. **対称性の合う軌道を組み合わせる**
   - σ軌道：s-s、s-pz、pz-pz
   - π軌道：px-px、py-py

3. **エネルギー差を考慮した相互作用**
   - エネルギーが近い軌道同士は強く相互作用
   - エネルギー差が大きいと相互作用は弱い

4. **分子軌道の形成**
   - 結合性軌道：より電気陰性な原子に偏る
   - 反結合性軌道：より電気陽性な原子に偏る

## 具体例：HF分子

```
H (1s) --- σ* (反結合性)
    \    /
     \  /
      \/
      /\
     /  \
    /    \
F (2s,2p) --- σ (結合性) - 主にF側に局在
         --- π (非結合性) - F 2px, 2py
```

重要なポイント：
- 電気陰性度の差により、電子密度は偏る
- 非結合性軌道が存在する場合がある
- HOMO-LUMOギャップで反応性を予測
